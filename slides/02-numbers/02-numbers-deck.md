# Python hat 3 Typen für Zahlen

::: incremental
1. Ganze Zahlen $\mathbb{Z}$
   - Integer
   - 1, 5, 10, 2022, -314, 123456789
2. Reelle Zahlen $\mathbb{R}$
   - Float
   - 1.0, -1.0, 5.0, 10.99997
   - 3.141592 $\approx\Pi$
   - 1.414213 $\approx\sqrt{2}$
3. Komplexe Zahlen $\mathbb{C}$
:::

--------------------------------------------------------------------------------
